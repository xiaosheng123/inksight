# 按键说明

当前固件默认只有一个配置按键，用来在**常规显示模式**和**重新进入配置流程**之间切换。

## 1. 当前支持的按键动作

### 单击

- 轻按一次配置按键，会在 **Live 模式** 和 **普通定时刷新模式** 之间切换。
- 固件里把短按判定为 **不少于 50ms、少于 2 秒**。

### 长按

- 长按约 **2 秒** 后，设备会先显示 `Restarting`，随后重启。
- 想进入重新配网/重新配置，请在重启过程中**继续按住按键片刻**，设备启动后会直接进入配置门户（Captive Portal）。

## 2. 两种运行模式是什么意思

### 普通定时刷新模式

- 设备按你在 Web 配置中设置的刷新周期工作。
- 大多数情况下更省电，适合日常桌面摆放。

### Live 模式

- 设备保持联网，按更短的轮询周期检查是否有待执行更新。
- 适合你正在调试模式、频繁改配置、想更快看到变化的时候。
- 代价是功耗会明显高于普通定时刷新模式。

## 3. 开机时如何强制进入配置门户

- 如果设备还没有 Wi-Fi 配置，开机会自动进入配置门户。
- 如果你已经配过网，也可以在开机时按住配置按键，设备会直接进入配置门户。
- 这个逻辑通常用于：更换 Wi-Fi、修改后端地址、救援无法正常联网的设备。

## 4. 当前版本没有哪些按键动作

- 当前固件**没有单独定义双击动作**。
- 当前固件也**没有“短按切下一模式”**这一行为；短按只负责切换 Live / 普通定时刷新模式。
